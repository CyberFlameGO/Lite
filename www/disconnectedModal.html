Troubleshooting

  • If the disconnect has occured immediately after loading Lite …

    This is often caused by AWSs caching, and Lite is trying to reconnect to an old server instance. Until we find a better solution, here are some ways to solve this issue:

    • Try refreshing the page
    • Try in a different browser, or in a private window
    • Try clearing your cookies/website data for this page (lite.docker.stat.auckland.ac.nz)
    • Contact us if you still have no success
  • If the disconnect occured after you uploaded some data …

    There may be an issue with the dataset, or Lite is running into trouble for some other reason.

    • If you haven't already, try loading the data again
    • If you are on a Mac, ensure you are not trying to upload a Numbers (.numbers) file, as Lite cannot read these
    • Try a different dataset. If you get the same issue, it might be an issue with Lite. In that case, please contact us.
  • If the disconnect occured after you performed some action …

    This is likely an issue with Lite. Please contact us and quote the above session ID.
